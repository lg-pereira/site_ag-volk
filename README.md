# Website Agência Volk

Landing page comercial desenvolvida para a **Agência Volk**, com foco em captação de leads, geração de autoridade e conversão de empresários interessados em gestão de tráfego, automação com IA e estrutura de performance digital.

## Tech Stack

- React como framework principal da aplicação.
- React Router 7.15.1 para estrutura de navegação e rotas.
- Tailwind CSS para estilização e construção rápida da interface.
- Framer Motion para animações e transições visuais.
- Lucide para iconografia moderna e consistente.
- Google Font API para carregamento tipográfico.
- Cloudflare como CDN e camada de entrega.

## Desenvolvimento

O projeto foi construído como uma landing page em React voltada para performance, com estrutura visual moderna, navegação objetiva e seções pensadas para leitura rápida e tomada de decisão. A combinação entre Tailwind CSS, Framer Motion e React Router favorece escalabilidade, manutenção e experiência fluida na navegação.

## Construção da página

A página foi desenhada com lógica de **vendas**, conduzindo o visitante por uma jornada de conversão:

1. Hero com promessa agressiva de resultado e CTA imediato.
2. Bloco de dor, mostrando perdas com tráfego sem estratégia.
3. Apresentação da proposta de valor e diferenciais da agência.
4. Seção de benefícios com foco em previsibilidade, automação e retorno.
5. Bloco de ecossistema de performance, detalhando anúncios, IA e tracking.
6. Cases e métricas para prova de resultado.
7. Depoimentos para reforço de confiança.
8. Fechamento com CTA final para consultoria.

## Objetivo do projeto

Transformar visitantes em leads qualificados e reuniões comerciais, usando copy direta, prova social, argumentação de dor e reforço de resultado financeiro. O site posiciona a Agência Volk como uma operação orientada a performance, não apenas como prestadora de anúncios.

## Estrutura da Copy

A construção da landing page segue princípios clássicos de conversão:

- Headline forte com apelo emocional.
- CTAs distribuídos ao longo da navegação.
- Quebra de objeções com linguagem simples e direta.
- Blocos de autoridade, números e provas reais.
- Oferta central baseada em resolução de dores, não em tarefas técnicas.

